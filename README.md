# Sitio web de encargo para cliente "Frente Patria"

El sitio es una SPA (Single Page Application).
Cuenta con diseño responsivo para pc, tablet y móvil.

## Estructura:
### Menu de Navegación
- Home.
- Biblioteca.
- Videoteca.
- Fotocopiadora.
- Ayudantes.
- Información estudiantil.
